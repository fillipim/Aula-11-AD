Aula Git e Github

git init

Inicializar um novo repositório Git em um diretório.

git status

Exibe o status atual do seu repositório. Mostra quais arquivos foram modificados, adicionados e excluídos.

git add . ou git add 'nome do arquivo'

Adiciona as alterações de um arquivo, preparando-o para o próximo commit

git commit

Grava uma alteração do diretório. A menssagem do commit deve descrever as alterações realizadas.

git log

Mostra o histórico de commits do repositório, exibindo informações como autor, data e menssagem

git checkout -b 'nome da branch'

Cria uma nova branch a partir da atual.

git checkout 'nome da branch'

Muda de branch.

git pull origin "nome da branch"

Puxa alterações da branch remota.

git push origin 'nome da branch atual'

Sobe alterações da branch atual para o repositório remoto.
